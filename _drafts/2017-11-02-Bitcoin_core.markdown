## Bitcoin core 설치하기
---

### 설치환경
- ubuntu 16.04


```sh
# Install gcc
$ sudo apt-get install build-essential automake pkg-config libevent-dev bsdmainutils

# Install OpenSSL
$ sudo apt-get install libtool autotools-dev autoconf
$ sudo apt-get install libssl-dev

# Install Boost 
$ apt-get install libboost-all-dev



```
